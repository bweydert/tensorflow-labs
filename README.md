# tensorflow-labs

## AWS

Recommend running with the `deeplearning` AMI, which is a fork of Amazon's DL AMI.

Start a new EC2 Instance as follows:
  * AWS us.east region
  * p2.xlarge instance type.
  * make sure network has public IP address
  * `deeplearning` AMI
  *  `hi.pem` key
  * `ubuntu` security group.
